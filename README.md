# nilufereksi.github.io

This is a simple starter portfolio site for Nilufe Reksi.

How to use:
1. Customize `index.html` with your bio, projects, and links.
2. Edit `assets/css/styles.css` for colors and fonts.
3. Publish to GitHub Pages by creating a repository named `nilufereksi.github.io` and pushing these files to the default branch.
